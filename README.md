## Documentation

See [http://vue-scaff.joenix.com/](http://vue-scaff.joenix.com/)

### Get started with Vue-Scaff

```sh
# install
yarn global add @vue/cli
# or
npm i -g @vue/cli

# create an application
vue create -p vue-scaff/vue-cli-preset --clone <project-name>

# start coding
yarn serve

# build project
yarn build
```
